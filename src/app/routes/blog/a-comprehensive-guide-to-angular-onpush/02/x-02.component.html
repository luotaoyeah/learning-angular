当 component 的 change detection 设置为 ChangeDetectionStrategy.OnPush 时, 只有在某些情况下, angular 才会触发 change detection:

  1. 当该 component 的某个 input property 的 reference 发生变更时
  2. 当该 component 或者它的 children 中触发了某个 DOM 事件时
  3. 手动触发:
    • 调用 ChangeDetectorRef.detectChanges() 方法触发该 component 及其下级 component 的 change detection cycle
