对 change detection 来说, 有两个重要的组成部分: component view 和 bindings, 当 angular 在编译 component 的 template 时, 会生成一个对应的 view, view 里面的节点树对应了 template 中的节点树

同时, 对于 view 中的每一个节点, angular 会分析并生成对应的 bindings 信息, 每一个 binding 信息实际上就是一个键值对, 其中的 key 就是绑定的属性, 其中的 value 就是绑定的表达式

所以, 前一个例子中的 binding 信息就是: 

{{ '{' }}
  name: 'textContent',
  expression: 'time | date: "HH:mm:ss.SSS"'
{{ '}' }}
