@angular-devkit/schematics-cli 是一个单独的 package, 是一个专门用来开发 schematics 相关功能的 command-line tool, 我们可以全局安装它, 然后使用命令 schematics blank --name=hello-world 创建一个新的 schematic collection, 并同时创建一个名为 hello-world 的 schematic

使用命令 schematics <path>:<schematic-name> 来执行一个 schematic, 其中 path 为 package.json 所在的目录

在 schematics project 目录下, 使用命令 schematics blank --name=hello-world-2 添加一个新的 schematic 到当前这个 collection 中
