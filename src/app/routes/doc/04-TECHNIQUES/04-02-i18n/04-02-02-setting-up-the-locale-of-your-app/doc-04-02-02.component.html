所谓 locale 指的是语言地区标识, 通常也称为 local identifier 或者 locale id

angular 中使用的 locale id 遵循的是 BCP47 规范, 它大概的一个描述规则是: language-extlang-script-region-variant-extension-privateuse 语言文字种类-扩展语言文字种类-变体-使用区域-方言-扩展-私有

angular 默认的 locale 是 en-US, angular 所有支持的 locale 列表可以查看: https://github.com/angular/angular/tree/master/packages/common/locales

简体中文的 locale 为 zh-Hans
