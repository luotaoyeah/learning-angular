angular 中的 data binding 根据数据流向可以分为三类：

  • source → view
    • interpolation
    • property
    • attribute
    • class
    • style
  • source ← view
    • event
  • source ↔ view

除了 interpolation 之外，其他绑定都有一个绑定目标，这个目标是一个 property

HTML attribute 和 DOM property 的区别：

  • HTML attribute 是在 HTML 中定义的，DOM property 是在 DOM 中定义的
  • attribute 和 property 不是一一对应的，但是大部分是对应的
  • attribute 是用来初始化 property 的，完了之后，attribute 的任务就完成了，它就不会再改变了；property 的值可以不断变化；用一句话来概括：attribute 是初始值，property 是当前值；
  • angular 中的 data binding 绑定的是 property 而不是 attribute
