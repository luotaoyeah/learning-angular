对于自定义的 component，我们可以通过 x/xChange 模式来实现 two-way binding，而对于内置的 HTML 标签，angular 提供了 NgModel 指令来实现 two-way binding，使用 NgModel 指令需要引入 FormsModule 模块

实际上 NgModel 指令也是一个语法糖，它包含 ngModel/ngModelChange 两个属性

如果没有 NgModel 指令，如果我们想自己实现 two-way binding，那么对于每种不同的表单元素，我们需要使用不同的处理逻辑，比如对于 <input type="text">
