property binding 可以使用 [property]="value" 的语法形式

property binding 也可以使用 bind-property="value" 的语法形式

property binding 绑定的目标可以是 element property 或者 directive input property, angular 会首先检查是否存在对应的 directive input property, 再检查是否存在对应的 element property, 如果这两种属性都找不到, 则会报错

如下, 第一个绑定的是 HTMLParagraphElement.textContent 这个 property, 第二个绑定的是 Doc0303020702Directive.textContent 这个指令属性

如果 propery binding 外面没有使用 [] 包裹, 则首先会检查是否存在对应的 directive input property, 再检查是否存在对应的 element attribute

如下, 第一个绑定的是 textContent 这个 attribute, 第二个绑定的是 Doc0303020702Directive.textContent 这个指令属性
