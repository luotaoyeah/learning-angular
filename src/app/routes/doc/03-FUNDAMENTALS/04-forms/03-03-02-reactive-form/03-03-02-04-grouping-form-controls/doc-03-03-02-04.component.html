在 template 中, 在 <form> 标签上通过 FormGroupDirective(formGroup) 指令绑定我们在 component class 中定义的 FormGroup 对象

然后在 <form> 里面的 <input> 标签上, 通过 FormControlName(formControlName) 指令绑定到 FormGroup 对象中的某个 FormControl 对象

我们可以在 <form> 中添加一个 submit 类型的 <button>, 当点击这个按钮时, 会自动触发 <form> 的 submit 事件, FormGroupDirective 指令会监听到 <form> 的 submit 事件, 然后发出一个 ngSubmit 事件, 然后我们再监听 <form> 上的 ngSubmit 事件, 来处理表单提交
