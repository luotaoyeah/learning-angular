injector 根据层级关系可以分为以下几种：

  • platform injector：

    内部使用，一般不会对它进行配置，它是 app root injector 的 parent injector

  • root injector

    整个应用的顶级 injector，实际上就是 AppModule 的 injector，可以在 AppModule 中使用 @NgModule.providers 进行配置，也可以直接在 service 中使用 @Injectable.providedIn: 'root' 进行配置

  • module injector

    NgModule 层级的 injector，可以通过 @NgModule.providers 进行配置，也可以直接在 servide 中使用 @Injectable.providedIn: SomeModule 进行配置；如果是一个 lazy-loaded module，则应该通过 @NgModule.providers 进行配置，否则的话这个 service 可能会被 tree-shake 掉

  • component injector

    component 层级的 injector，可以通过 @Component.providers 进行配置

    component injector 只能对该 component 以及它的下级 component 进行注入，不能对其他地方进行注入
