angular.json#projects 下面每个 project 包含以下一些配置：

  • root 该 project 的根目录，默认项目的该属性为空，因为默认项目位于 workspace 的根目录下
  • sourceRoot 项目 source files 的根目录，默认为 src
  • projectType 项目类型，可以是 application 或者 library
  • prefix 使用 ng 命令自动生成 schematics 时的 selector 的前缀，默认为 app
  • schematics 针对该 project 的 schematics 的默认值配置，可以覆盖针对 workspace 的 schematics 的默认配置
  • architect 针对构建的相关配置
