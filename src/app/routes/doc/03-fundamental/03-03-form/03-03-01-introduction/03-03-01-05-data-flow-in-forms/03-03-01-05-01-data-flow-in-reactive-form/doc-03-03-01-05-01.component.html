在 reactive form 中，以 text input element 为例，数据从 view 流向 model 的一个过程是：

  • 用户输入文本，text input element 触发 input 事件
  • control value accessor 监听到 input 事件，触发 FormControl 实例调用 FormControl.setValue() 方法更新数据
  • FormControl 实例通过 valueChanges 这个 observable 推送最新数据，所有订阅了 valueChanges 的对象都可以接收到数据

数据从 model 流向 view 的一个过程是：

  • 调用 FormControl 实例的 setValue() 方法
  • valueChanges 这个 observable 推送数据
  • 所有订阅了 valueChanges 的对象都可以接收到数据
  • control value accessor 负责将新的数据更新到 input element 中去
