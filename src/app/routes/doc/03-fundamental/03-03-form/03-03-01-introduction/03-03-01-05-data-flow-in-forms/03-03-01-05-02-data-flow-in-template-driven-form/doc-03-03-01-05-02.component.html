在 template-driven form 中，数据从 view 流向 model 的一个过程是：

  • 用户输入，触发 input 事件
  • control value accessor 监听到 input 事件，然后调用 FormControl.setValue() 方法
  • FormControl 对象通过 valueChanges 这个 observable 推送新的数据
  • control value accessor 在调用 FromControl.setValue() 方法的同时，出发了 NgModel.viewToModelUpdate() 方法，从而触发了 ngModelChange 事件，从而更新了对应的 property

在 template-driven form 中，数据从 model 流向 view 的一个过程是：

  • 在 component 中更新了 property
  • 触发了 change detection
  • NgModel 指令的 ngOnChanges() 方法执行
  • 触发 FormControl 对象更新数据
  • FormControl 通过 valueChanges 这个 observable 推送数据
  • control value accessor 更新 input element 的值
