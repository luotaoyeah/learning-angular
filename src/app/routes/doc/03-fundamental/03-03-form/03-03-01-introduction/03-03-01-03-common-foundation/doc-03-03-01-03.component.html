reactive form 和 template-driven form 有一些共享的概念和对象：

  • FormControl
    • tracks the value and validation status of an individual form control
  • FormGroup
    • tracks the values and validation status of a collection of FormControls
  • FormArray
    • tracks the values and validation status of an array of FormControls
  • ControlValueAccessor
    • creates a bridge between FormControl instance and DOM element
