reactive form 和 template-driven form 有一些共享的概念和对象：

  • FormControl
    • 追踪单个 FormControl 的 value 和 validation 状态
  • FormGroup
    • 追踪一组 FormControl 的 value 和 validation 状态
  • FormArray
    • 追踪一个 FormControl 列表的 value 和 validation 状态
  • ControlValueAccessor
    • 在一个 FormControl 对象实例和一个 DOM 元素之间建立桥梁
