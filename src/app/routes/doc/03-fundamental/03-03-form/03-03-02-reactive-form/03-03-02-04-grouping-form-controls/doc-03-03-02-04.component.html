在 template 中，在 <form> 标签上通过 FormGroupDirective 指令的 formGroup 属性绑定我们在 component class 中定义的 FormGroup 对象

然后在 <form> 里面的 <input> 标签上，通过 FormControlName 指令的 formControlName 属性绑定到 FormGroup 对象中的某个 FormControl

我们可以在 <form> 中添加一个 submit 类型的 <button>，当点击这个按钮时，会自动触发 <form> 的 submit 事件，FormGroupDirective 指令会监听 <form> 的 submit 事件，然后 emit 一个 ngSubmit 事件，然后我们再监听 ngSubmit 事件，来处理表单提交
