直接将 service class 作为 providers 的元素，实际上是一种简写的方式，正常的方式如下：

  // 简写方式
  providers: [ SomeService ]

  // 正常方式
  providers: [ {{ '{' }} provide: SomeService, useClass: SomeService } ]

providers 的元素是一些 Provider 对象，其中 provide 属性用来指定 token，useClass 属性表示使用 service class 作为 provider

除了使用 useClass 之外，还可以使用其他的方式来定义 provider，比如 useFactory，useExisting，useValue
