angular 中的 projects 有两种类型: application 和 library, 可以通过 angular.json#projects.learning-angular.projectType 属性配置类型

如何创建一个 library 类型的 project ? 使用命令: `ng generate library <library-name>`

angular application 的 builder 使用的是: @angular-devkit/build-angular:browser, angular library 的 builder 使用的是: @angular-devkit/build-ng-packagr:build, 这个 builder 的一个特性就是不需要指定 --prod, 自动就会使用 aot 进行编译
