在 reactive form 中, 以 <input type="text"/> 为例, 数据从 view 流向 model 的一个过程是:

  1. 用户在 <input /> 中输入文本
  2. <input /> 触发 input 事件
  3. control value accessor 监听到 input 事件, 将最新数据传给 FormControl 实例
  4. FormControl 实例通过 valueChanges 这个 observable 推送最新数据
  5. 所有订阅了 valueChanges 的订阅者都会接收到最新数据

数据从 model 流向 view 的一个过程是:

  • 调用 FormControl.setValue() 方法, 更新 FormControl 中的数据
  • FormControl 通过 valueChanges 这个 observable 推送数据
  • 所有订阅了 valueChanges 的对象都会接收到数据
  • control value accessor 负责将新的数据更新到 <input /> 中去
