angular 官方定义的一些常用的 NgModule 有：

  • BrowserModule（@angular/platform-browser）：提供了浏览器相关的功能，如果应用要运行在浏览器中，则需要引入该模块
  • CommonModule（@angular/common）：提供一些通用的功能，比如 *ngIf，*ngFor 等指令
  • FormsModule（@angular/forms）：template-driven form 相关功能，包括：NgModel 指令
  • ReactiveFormsModule（@angular/forms）：reactive form 相关功能，包括：FormGroup，FormControl，FormArray
  • RouterModule（@angular/router）：提供路由相关的功能
  • HttpClientModule（@angular/common/http）：提供 http 相关功能，包括：HttpClient 服务

如果使用一个 NgModule ？在其他的 NgModule 的 metadata 中的 imports 中声明这个 NgModule 就可以了，angular 就知道怎么去加载这个 NgModule 了

BrowserModule 只能在 root module（AppModule） 中引入，不能在其他的下级 NgModule 中引入，否则编译报错；同时 BrowserModule 引入了 CommonModule 并且 re-export 了 CommonModule
