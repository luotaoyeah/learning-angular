Overview

angular 的 router 是直接借鉴了浏览器的导航模型：

  • 通过解析 URL 地址，加载对应的视图
  • URL 中可以传递参数
  • 可以通过点击连接进行跳转，也可以使用命令式（imperative）的方式，手动触发跳转
  • 可以点击前进/后退按钮，在历史记录之间进行跳转

The Basics

Router Imports

router 相关的功能位于 @angular/router 这个模块下面，比如 RouterModule，Routes

RouterOutlet

RouterOutlet 是一个 directive，用法类似于一个 component：<router-outlet></router-outlet>，用来指示要将 component 渲染到哪个地方？类似于 vue 中的 <router-view></router-view>

当某个 component 被渲染到对应的 router-outlet 处时，实际上这个 component 和 router-outlet 是作为兄弟元素同时存在的，这一点跟 vue 中的 router-view 是不一样的，在 vue 中，渲染的组件会替换掉 router-view，而不是同时存在

Router Links

RouterLink（routerLink） 是一个 directive，用来将一个 <a> 标签标识为一个路由链接，点击这个链接，就会跳转到指定的地址
