property binding 可以使用 [someProperty]="someExpression" 的形式, 也可以使用 someProperty="someValue", 它们的效果是不一样的

当使用 [someProperty]="someExpression" 的形式时, someExpression 表示的是一个表达式, angular 会对它进行解析

并且, 当 [someProperty] 不能被识别为一个 directive input property 时, 它会被尝试识别为一个 element property

当使用 someProperty="someValue" 的形式时, someValue 是一个字符串, angular 不会对它进行解析

并且, 当 someProperty 不能被识别为一个 directive input property 时, 它会被尝试识别为一个 element attribute
