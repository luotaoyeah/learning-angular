what are generation schematics?

schematics 是一系列用来修改 project 的 instructions, 通过 add files 或者 modify existing files 的方式

默认提供给 ng generate 命令的 schematics 是放在 @angular 包里面的, 而他们的 JSON schema 是放在 @schematics/angular 包里面的, 默认的 schematics 配置包括：

  • @schematics/angular:component
  • @schematics/angular:directive
  • @schematics/angular:module
  • @schematics/angular:service
  • @schematics/angular:pipe
  • @schematics/angular:class

每一个 schematics 配置下面的每一个配置选项, 就对应该 ng generate 命令后面的一个参数, 配置了之后, 在使用 ng generate 命令的时候, 这些配置就会作为这些参数的默认值
