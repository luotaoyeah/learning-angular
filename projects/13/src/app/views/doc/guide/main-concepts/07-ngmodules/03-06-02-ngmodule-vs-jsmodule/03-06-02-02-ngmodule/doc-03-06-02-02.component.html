NgModule 实际上就是一个 JS class，然后使用 @NgModule 装饰器进行配置，同一个 JS 文件中可以包含多个 NgModule

declarables 指的是 component，directive，pipe，这三种类型的数据可以用在 NgModule 的 declarations 配置中，表示它们是属于这个 NgModule 的资源，可以在这个 NgModule 下面所有的 template 中使用，即： declarations 用来声明当前 NgModule 中定义了哪些 declarables

imports 用来引入其他的 NgModule，引入之后，这些引入的 NgModule 中 exported 的 declarables，就可以在当前 NgModule 中使用了，即：imports 引入的是 NgModule，但是实际上引入的是这些 NgModule 输出的 declarables

NgModule 可以通过 exports 将自己的 declarables 输出，这样其他的 NgModule 引入这个 NgModule 之后，就可以使用这些 declarables 了；同时，exports 也可以输出某些当前 NgModule 引入的 NgModule，相当于将这些 NgModule 输出的 declarables 再次输出（re-export）

一个 declarable（component，directive，pipe）只能在一个 NgModule 中定义，不能在多个 NgModule 中定义，否则编译报错
